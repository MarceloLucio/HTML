Propriedades de CSS

Propriedades usadas para estilizar o site. Android

Propriedades Descrição
@charset A regra de CSS especifica a codificação de caracteres usada na folha de estilo. Deve ser o primeiro elemento da folha de estilo e não ser precedido por nenhum caractere; como não é uma instrução aninhada , não pode ser usada dentro do grupo condicional at-rules . Se forem definidas várias at-rules, apenas a primeira é usada, e não pode ser usada dentro de um atributo em um elemento HTML ou dentro do elemento onde o conjunto de caracteres da página HTML é relevante.
@import O CSS at-rule é usado para importar regras de estilo de outras folhas de estilo.
@font-face A regra de CSS especifica uma fonte personalizada com a qual exibir o texto; a fonte pode ser carregada de um servidor remoto ou de uma fonte instalada localmente no próprio computador do usuário.
:root A pseudo-classe CSS corresponde ao elemento raiz de uma árvore que representa o documento. Em HTML, representa o elemento e é idêntico ao seletor, exceto que sua especificidade é maior.
:hover A pseudoclasse CSS corresponde quando o usuário interage com um elemento com um dispositivo apontador, mas não necessariamente o ativa. Geralmente é acionado quando o usuário passa o mouse sobre um elemento com o cursor (ponteiro do mouse).
::after Em CSS, ::after cria um pseudoelemento que é o último filho do elemento selecionado. Geralmente é usado para adicionar conteúdo cosmético a um elemento com a contentpropriedade. É em linha por padrão.
font-family A font-family propriedade CSS especifica uma lista priorizada de um ou mais nomes de família de fontes e/ou nomes de família genéricos para o elemento selecionado.
font-size A propriedade CSS define o tamanho da fonte. Alterar o tamanho da fonte também atualiza os tamanhos das unidades relativas ao tamanho da fonte, como , e assim por diante.
font-weight A propriedade CSS define o peso (ou negrito) da fonte. Os pesos disponíveis dependem do que está definido no momento
text-align A propriedade CSS define o alinhamento horizontal do conteúdo de nível embutido dentro de um elemento de bloco ou caixa de célula de tabela. Isso significa que funciona como , mas na direção horizontal.
text-indent A text-indent propriedade CSS define o comprimento do espaço vazio (recuo) que é colocado antes das linhas de texto em um bloco.
text-decoration A propriedade CSS abreviada define a aparência das linhas decorativas no texto. É uma abreviação para , , e a propriedade mais recente.
line-height A propriedade CSS define a altura de uma caixa de linha. É comumente usado para definir a distância entre as linhas de texto. Em elementos de nível de bloco, especifica a altura mínima das caixas de linha dentro do elemento. Em elementos embutidos não substituídos , ele especifica a altura que é usada para calcular a altura da caixa de linha.line-height
background-color A propriedade CSS define a cor de fundo de um elemento.
background-image A propriedade CSS define uma ou mais imagens de fundo em um elemento.
background-size A propriedade CSS define o tamanho da imagem de fundo do elemento. A imagem pode ser deixada em seu tamanho natural, esticada ou restringida para caber no espaço disponível.
background-repeat propriedade CSS define como as imagens de fundo são repetidas. Uma imagem de fundo pode ser repetida ao longo dos eixos horizontal e vertical, ou não ser repetida.
background-position A propriedade CSS define a posição inicial para cada imagem de fundo. A posição é relativa à camada de posição definida por.
background-attachment A propriedade CSS define se a posição de uma imagem de plano de fundo é fixa na viewport ou rola com o bloco que a contém.
height A height propriedade CSS especifica a altura de um elemento. Por padrão, a propriedade define a altura da área de conteúdo . Se box-sizingestiver definido como border-box, no entanto, ele determina a altura da área da borda.
width A width propriedade CSS define a largura de um elemento. Por padrão, ele define a largura da área de conteúdo , mas se box-sizingestiver definido como border-box, ele define a largura da área da borda.
padding A propriedade abreviada CSS define a área de preenchimento em todos os quatro lados de um elemento de uma só vez.
margin A propriedade abreviada CSS define a área da margem em todos os quatro lados de um elemento.
border-radius A propriedade CSS arredonda os cantos da borda externa de um elemento. Você pode definir um único raio para fazer cantos circulares ou dois raios para fazer cantos elípticos.
box-shadow A propriedade CSS adiciona efeitos de sombra ao redor do quadro de um elemento. Você pode definir vários efeitos separados por vírgulas. Uma sombra de caixa é descrita por deslocamentos X e Y em relação ao elemento, raio de desfoque e dispersão e cor.
postion A propriedade CSS define como um elemento é posicionado em um documento. As propriedades , , e determinam a localização final dos elementos posicionados.
Left / Top A propriedade CSS participa da especificação da posição horizontal de um elemento posicionado . Não tem efeito em elementos não posicionados.
transform A propriedade CSS permite girar, dimensionar, inclinar ou traduzir um elemento. Ele modifica o espaço de coordenadas do modelo de formatação visual CSS .
display Uma propriedade CSS display especifica o tipo de caixa de renderização usada por um elemento. No HTML, os padrões da propriedade display são feitos a partir do comportamento descrito nas especificações HTML ou da folha de estilo padrão do navegador/usuário. O valor padrão em XML é inline.
list-style-type A propriedade CSS define o marcador (como um disco, caractere ou estilo de contador personalizado) de um elemento de item de lista.
list-style-position A propriedade CSS define a posição do relativo a um item de lista.
columns A propriedade abreviada CSS define o número de colunas a serem usadas ao desenhar o conteúdo de um elemento, bem como as larguras dessas colunas.
content A propriedade CSS substitui um elemento por um valor gerado. Objetos inseridos usando a propriedade são elementos substituídos anônimos.
